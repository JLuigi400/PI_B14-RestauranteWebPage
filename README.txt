=========================================
README - Salud Juárez
=========================================

Este proyecto es una plataforma web para la gestión de Restaurantes y Proveedores de Ciudad Juarez, inicialmente. Con el objetivo de mostrar diferentes restaurantes poco conocidos.

Objetivo Inicial - 
Ayuda a los usuarios a ubicar diferentes restaurantes y proveedores de manera facil y rapida filtrando distintos parametros, sea por ubicación, tipo de comida, precios, etc. Incluyendo si es saludable, posibles alergenos, etc.

Trabajo Universitario de la UACJ - Universidad Autonoma de Ciudad Juarez - 2026
